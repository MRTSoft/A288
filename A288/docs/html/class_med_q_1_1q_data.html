override string MedQ.qData.ToString ( )

The string follows the pattern:
Question text <ENTER>
ans[0]<ENTER>
ans[1]<ENTER>
ans[2]<ENTER>
ans[3]<ENTER>
ans[4]<ENTER>

Note
The # before ans[i] marks the answer as true. If the answer is false the # will not appear.
Returns
The string representing the qData object that follows the pattern used for .txt files.
